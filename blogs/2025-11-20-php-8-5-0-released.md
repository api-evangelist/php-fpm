The PHP development team announces the immediate availability of PHP 8.5.0. This release marks the latest minor release of the PHP language. PHP 8.5 comes with numerous improvements and new features such as: New "URI" extension New pipe operator (|>) Clone With New #[\NoDiscard] attribute Support for closures, casts, and first class callables in constant expressions And much much more... For source downloads of PHP 8.5.0 please visit our downloads page ,
  Windows source and binaries can also be found there .
  The list of changes is recorded in the ChangeLog . The migration guide is available in the PHP Manual.
  Please consult it for the detailed list of new features and backward incompatible changes. Kudos to all the contributors and supporters!
